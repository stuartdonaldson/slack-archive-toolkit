# F3 Slack: Start Here

Cover, briefly:

1. Slack setup usually happens after the first workout and once the new guy has an F3 name.
2. Join the local Slack workspace for the AO or region where he first posted.
3. Join the AO channel; ask the PAX who invited him, the Q, or someone from the workout which channel to join.

   * Use the F3 name as the Slack display name.

5. Join F3 Puget Sound Slack after the local workspace:

   * Do not overstate formal governance or parent/child structure unless the digest explicitly supports that.
6. Keep it simple: start with the AO channel, the local region’s main/all channel, and F3 Puget Sound. Add more channels only when useful.
